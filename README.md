# 🚀 ADSD – Adaptive Drivable Space Detection

> Reimagining how machines understand the road ahead.

---

## 🌌 1. Overview

*Adaptive Drivable Space Detection (ADSD)* is not just a project — it's a step toward intelligent mobility. This system dynamically identifies safe drivable regions using advanced computer vision and machine learning techniques.

✨ Built for innovation. Designed for real-world impact.

## 🧠 2. Model Architecture

### ⚙️ Pipeline Flow (Animated Representation)

 <p align="center"> <img src="[https://media.giphy.com/media/26AHONQ79FdWZhAI0/giphy.gif](https://media.giphy.com/media/26AHONQ79FdWZhAI0/giphy.gif)" width="500"/> </p>

👉 **Flow:**
**Input → Preprocessing → Segmentation → Risk Analysis → Detection → Decision → Output**

### 🔹 Key Modules:

* 📥 Input (Camera / Video Feed)
* 🧹 Preprocessing (Resize, Normalize)
* 🧠 Segmentation Model (CNN-based)
* ⚠️ Risk Detection (Obstacles, edges)
* 🚦 Decision System
* 📤 Output Visualization

## ⚙️ 4. Setup & Installation Instructions

### 🖥️ Requirements:

* Python 3.8+
* pip
* (Optional) Node.js

### 🔧 Installation:

```
# Clone repository
git clone https://github.com/your-username/ADSD.git

# Move into project
cd ADSD

# Install dependencies
pip install -r requirements.txt
```

---

## ▶️ 5. How to Run the Code

🚀 **Run Main Program**

```bash
cd src
python main.py
```

🌐 **Run Web UI (Optional)**

```bash
npm install
npm run dev
```

💡 **Make sure:**

* Input file path is correct
* Dependencies are installed
