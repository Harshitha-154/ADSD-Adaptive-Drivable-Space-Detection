## 🛠️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/ADSD.git

# Navigate into the project
cd ADSD

# Install dependencies
npm install

# Start development server
npm run dev
```

💡 Ensure you have Node.js and npm installed.
